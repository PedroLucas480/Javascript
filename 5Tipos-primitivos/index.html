tipos primitivos

vamos entender o que são tipos primitivos

eles são

  • string

    • tipo string o tipo string do javascript é usado para representar dados textuais.cada elemento na string ocupa uma posição na string.o positivos e negativos.

    ex:azul,carlos,idade.

  • number

    • o tipo number é capaz de armazenar números de ponto flutuante positivos e negativos.

    ex:7,15,-8,1.5,0.2,7.9

  • boolean

    • boolean representar uma entidade logica e pode ter dois valores:true e false.

    ex:true false

  • null

    • o tipo null tem exatamente um valor:null valor nulo ou vazio.

  • undefined

    • uma variavel que não recebeu um tipo definitivo,undefined ou seja indefinido

para identificar o tipo da variavel utilizamos o camando(typeof)+ a variavel
